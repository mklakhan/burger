# Eat-Da-Burger

### Overview

Eat-Da-Burger app with MySQL, Node, Express, Handlebars and ORM, following the MVC design pattern and using Node and MySQL to query and route data, and Handlebars to generate HTML.

The app lets users input the names of burgers they'd like to eat and then devour them! 


App can be accessed at : https://fast-bastion-78443.herokuapp.com/
